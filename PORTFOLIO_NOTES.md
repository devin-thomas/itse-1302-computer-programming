# Portfolio Notes

This repository documents my first programming class: ITSE 1302, taken online at
San Antonio College in Summer 2023.

## Why This Repo Matters

The value of this repository is not that the code is advanced. Its value is that
it captures the beginning of the learning curve: writing first scripts, asking
for user input, using variables, making decisions with conditionals, repeating
work with loops, and organizing simple programs with functions.

Keeping this work visible makes later projects easier to understand. A portfolio
should show progress, and this repo is the baseline.

## What Is Preserved

- Original Python coursework files
- Chapter-by-chapter exercise organization
- Standalone scripts completed during the course
- Final project screenshots and source file

## What I Would Improve Now

Looking back with more experience, the next improvements would be to add more
consistent naming, input validation, reusable helper functions, automated tests,
and clearer separation between program logic and console interaction.

Those changes are intentionally not made here. This repository is kept as an
honest early-career snapshot, with documentation added around it for portfolio
readability.
