# ITSE 1302 - Computer Programming

Coursework from **ITSE 1302: Computer Programming**, taken online at
**San Antonio College** during **Summer 2023**.

This repository is part of my development portfolio. It preserves the work from
my first programming class as a starting-point record: small Python exercises,
chapter assignments, and a final console project. The code is intentionally kept
close to its original beginner-course form so future projects can show clear
growth over time.

## Running the Programs

These are standalone Python scripts and do not require third-party packages.
From the repository root, run any file with Python 3:

```powershell
python "Chapter 2/salestax.py"
python finalproject/rps.py
```

Some assignments prompt for console input. The final project also expects typed
input from the user.
